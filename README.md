# 📊 HR Analytics Dashboard

## 📌 Overview
This project is an HR Analytics Dashboard built to track and analyze candidate data collected through Google Forms. The dashboard provides insights into recruitment performance, candidate distribution, and hiring trends.

---

## 🔗 Data Source
- Data is collected using **Google Forms**
- Candidates fill their details directly
- Data is automatically stored in **Google Sheets**
- Dashboard is connected for real-time updates

---

## 🛠️ Tools Used
- Microsoft Excel / Power BI
- Google Forms
- Google Sheets
- Data Cleaning & Transformation

---

## 📈 Key Features
- 📌 Total Vacancy & Current Vacancy Tracking  
- 👨‍💼 Candidate Interview Data Analysis  
- 📊 Week-wise Continue vs Discontinue Analysis  
- 👥 Gender Distribution  
- 🎓 Qualification Analysis  
- 🧠 Experience Breakdown  
- 📍 Position-wise Application Count  
- 📋 Candidate Details Table
